Reducing deprecated API usage
in LLM code completion

A retrieval-augmented, finetuned code completion system that steers generations toward modern, non-deprecated APIs — combining a deprecated-API knowledge base with a finetuned DeepSeek-Coder model.

Try the playground → See example cases

Retrieval

Hybrid BM25 + dense retrieval over a deprecated-API knowledge base, fused with reciprocal rank fusion and reranked with a cross-encoder, surfaces relevant API-change context for the code being completed.

Finetuning

DeepSeek-Coder-1.3B-instruct finetuned to prefer modern API usage when completing code, compared against base-model and retrieval-only baselines.

Evaluation

Measured via Deprecated API Usage Rate (DUR) and Replacement Usage Rate (RUR), alongside ROUGE-L, Hit@3, MRR and NDCG@3 for retrieval quality.
